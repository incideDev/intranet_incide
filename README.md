# Intranet Incide (Reference Architecture)

Questo repository contiene la codebase della intranet PHP interna.

Scopo:
- riferimento architetturale
- pattern di implementazione
- esempi di servizi, views e routing

Non è pensato per essere installato direttamente.

## Stack

PHP
MySQL
Vanilla JS
Custom service architecture
